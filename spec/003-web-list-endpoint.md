### 端点管理页面
Tasks页面改成端点列表页面

服务名称、url、创建时间、当前状态（运行中、已停用）、当前连接数（stdio,sse,streamable）

#### 功能
* 分页查看端点列表
* 按服务名称或接口名称、当前状态筛选数据
* 查看端点详情
* 删除端点
* 编辑端点

##### 端点编辑页
> 点击编辑，弹出框endpoint编辑页

注意点：
* 服务名称：用户手填，不能为空
* method：POST\GET\PUT\DELETE\OPTIONS\HEAD\PATCH，按用户填写的swagger接口详情自动解析
* url: 按用户填写的swagger接口详情自动解析
* swagger接内容需高亮显示
```   
+----------------------------------------------------+
|     服务名称                                        |
|     method                                         |
|     url                                            |
|   +--------------------------------------------+   |
|   |                                            |   |
|   |                                            |   |
|   |                                            |   |
|   |        swagger接口详情（json或yaml格式）      |   |
|   |                                            |   |
|   |                                            |   |
|   +--------------------------------------------+   |
|                                        提交  关闭   |
+----------------------------------------------------+
```
